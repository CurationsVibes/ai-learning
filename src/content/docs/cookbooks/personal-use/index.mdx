## Quick Start Examples

### Budget Planning
```
Help me create a monthly budget template. Include categories for:
- Housing (rent/mortgage)
- Utilities
- Food (groceries + dining)
- Transportation
- Entertainment
- Savings
Format as a simple spreadsheet layout I can customize.
```

### Meal Planning
```
Create a 5-day dinner meal plan for a family of 4. Requirements:
- Budget-friendly ingredients
- 30 minutes or less prep time
- Vegetarian-friendly options
- Include a shopping list grouped by store section
```

### Email Draft
```
Help me write a professional but friendly email to reschedule a meeting. Key points:
- Apologize for the inconvenience
- Propose 2-3 alternative times next week
- Maintain positive tone
- Keep it brief
```
